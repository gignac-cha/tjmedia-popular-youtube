TJMedia Popular Worker

Available routes:

  • GET / or GET /help returns this document.
  • GET /search proxies the TJ Media topAndHot100 endpoint (with R2 caching).
    All parameters are optional:
    • chartType — TOP or HOT (default: TOP)
    • strType — 1 (가요), 2 (POP), 3 (J-POP) (default: 1)
    • searchStartDate — YYYY-MM-DD (default: 1st of current month)
    • searchEndDate — YYYY-MM-DD (default: today)
  • OPTIONS * returns CORS headers.
  • HEAD * returns an empty success response.
